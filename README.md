# Yktours-admin
Веб-интерфейс администрирования сервиса Yktours по приобретению путевок на теплоходе.

### Быстрый старт

Для запуска проекта в командной строке введите следующие команды:

        $ cd Yktours/admin       (Linux)
        cd Yktours\admin         (Windows)

        $ yarn start            (Linux)
        yarn start              (Windows)
